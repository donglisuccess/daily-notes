> 在学习投资和理财的过程中，我们经常会听到一个词：**量化投资**。很多人一听到"量化"，第一反应就是用 Python 写代码炒股、用 AI 自动赚钱。量化投资的本质不是"稳赚"，而是把投资逻辑变成明确的规则、指标和模型，然后用数据和程序去执行、验证和优化。

本文主要整理量化投资中常见的四大策略方向：

- 基本面量化策略
- 技术面量化策略
- 统计套利策略
- AI 驱动策略

> 这篇文章的目标不是让我们马上实盘赚钱，而是建立基础认知，为后续模拟交易、Python 回测和策略验证打基础。

---

## 一、基本面量化策略：买的是公司价值

### 1.1 什么是基本面策略？

基本面策略关注的是：**一家公司本身到底好不好？贵不贵？未来有没有赚钱能力？**

它不是重点看今天涨了还是跌了，而是看企业背后的经营质量和财务表现。比如：

| 分析维度 | 具体指标 |
|---|---|
| 成长性 | 营收增长、净利润增长 |
| 财务健康 | 负债率、现金流 |
| 估值 | PE、ROE 是否合理 |
| 行业前景 | 市场空间、竞争格局 |

> **生活化比喻**：如果你要投资一家饭店，基本面策略不是看今天门口排队人多不多，而是看这家饭店菜品好不好、成本控制怎么样、客流稳不稳定、老板会不会经营、利润率高不高。

所以，基本面策略更适合**中长期投资**。

### 1.2 基本面策略常见因子

#### 估值类因子

估值类因子主要用来判断股票贵不贵：

| 指标 | 含义 | 说明 |
|---|---|---|
| PE | 市盈率 | 越低越便宜，但可能代表成长性差 |
| PB | 市净率 | 适合金融类公司 |
| PS | 市销率 | 适合营收稳定的企业 |
| 股息率 | 现金分红回报 | 越高持有收益越高 |

> **注意**：低 PE 可能代表便宜，也可能代表市场认为它未来增长差。高 PE 可能代表贵，也可能代表市场认为它未来成长性强。估值指标不能单独看，要结合行业、成长性和盈利质量一起判断。

#### 成长类因子

成长类因子主要用来判断公司未来有没有增长潜力：

| 指标 | 说明 |
|---|---|
| 营收增长率 | 公司业务扩张速度 |
| 净利润增长率 | 实际赚钱能力增长 |
| 扣非净利润增长率 | 排除一次性收益的真实增长 |
| 毛利率变化 | 定价能力和成本控制 |
| 研发投入增长 | 技术型公司的成长潜力 |

#### 盈利类因子

盈利类因子主要看公司赚钱能力：

| 指标 | 含义 | 优秀标准 |
|---|---|---|
| ROE | 净资产收益率 | > 15% 为优质 |
| ROA | 总资产收益率 | > 5% 为良好 |
| 毛利率 | 销售收入中利润占比 | 越高越好 |
| 净利率 | 扣除所有成本后的利润率 | 稳定增长最佳 |
| 经营现金流 | 实际收到的现金 | 必须为正 |

> 如果一家公司的 ROE 是 20%，可以粗略理解为：**公司用 100 元净资产，一年能赚 20 元利润**。长期高 ROE 的公司，往往说明商业模式不错。但也要小心，有些公司 ROE 高是因为高负债堆出来的，所以还要结合资产负债率一起看。

#### 风险类因子

风险类因子主要用来避坑：

- 资产负债率过高 → 财务风险大
- 应收账款过多 → 可能存在坏账
- 经营现金流长期为负 → 赚钱能力存疑
- 商誉过高 → 容易暴雷
- 大股东质押比例高 → 可能有利益输送
- 利润波动过大 → 经营不稳定

> 风险类因子不一定帮我们找到上涨最快的股票，但可以帮助我们**避开明显有问题的公司**。

### 1.3 基本面策略示例

假设我们想做一个银行股筛选策略，可以设置规则：

```python
# 银行股筛选策略示例
pe < 8              # 市盈率小于 8
pb < 1              # 市净率小于 1
roe > 10            # 净资产收益率大于 10%
资产负债情况稳定    # 排除高负债公司
净利润连续 3 年增长 # 业绩稳定
股息率 > 4%         # 现金回报可观
```

然后用程序从所有银行股中筛选符合条件的公司。**这就是一个简单的基本面量化策略**——不是靠感觉选股，而是先设定规则，再用数据执行。

### 1.4 基本面策略的优缺点

| 优点 | 缺点 |
|---|---|
| 逻辑清晰，容易理解 | 财报数据更新慢，存在滞后性 |
| 适合中长期投资 | 短期交易能力较弱 |
| 和企业真实经营结合紧密 | 财报质量参差不齐 |
| 比较适合普通投资者入门 | 可能买到"低估值陷阱" |

> 一句话总结：**基本面策略适合回答"这家公司值不值得长期买"，但不擅长回答"明天会不会涨"。**

## 二、技术面量化策略：看价格和成交量找机会

### 2.1 什么是技术面策略？

技术面策略关注的是：**价格怎么走？成交量怎么变？市场情绪有没有形成趋势？**

它不重点研究公司好不好，而是研究市场交易行为。

| 常见数据 | 说明 |
|---|---|
| 价格数据 | 开盘价、收盘价、最高价、最低价 |
| 成交量 | 买卖活跃程度 |
| 技术指标 | 均线、RSI、MACD、波动率 |

> **生活化比喻**：基本面是在研究饭店本身好不好，技术面是在看饭店门口排队的人是越来越多，还是越来越少。如果一家店最近排队越来越长，说明市场热度上来了。

### 2.2 常见技术面策略

#### 趋势跟随策略

趋势跟随的核心思想是：**涨了可能继续涨，跌了可能继续跌**。

```python
# 趋势跟随策略示例
if 价格向上突破 50 日均线:
    买入  # 认为趋势可能变强
elif 价格向下突破 50 日均线:
    卖出  # 认为趋势可能变弱
```

> 这种策略适合趋势明显的行情，但在震荡行情中容易反复亏损。

#### 动量策略

动量策略的核心思想是：**最近表现强的资产，未来一段时间可能继续强**。

| 资产 | 过去 3 个月涨幅 | 动量信号 |
|---|---|---|
| A 股票 | +30% | 强 |
| B 股票 | +5% | 中 |
| C 股票 | -10% | 弱 |

> 动量策略可能会选择 A 股票，因为它近期更强。但风险是，一旦市场风格切换，强势股可能快速回调。

#### 均值回归策略

均值回归的核心思想是：**涨太多可能会跌回来，跌太多可能会涨回来**。

```python
# RSI 超买超卖策略
if RSI < 30:
    考虑买入  # 可能超卖，反弹概率大
elif RSI > 70:
    考虑卖出  # 可能超买，回调概率大
```

> **注意**：超跌不等于一定会反弹，有些股票下跌是因为基本面真的恶化。所以技术指标不能孤立使用。

#### 成交量策略

成交量策略主要看资金是否真的参与：

```python
# 放量突破信号
if 股价突破新高 and 成交量 > 20日均量 * 2:
    买入  # 上涨有资金推动，非偶然波动
```

### 2.3 技术面策略的优缺点

| 优点 | 缺点 |
|---|---|
| 数据实时性强 | 容易过拟合 |
| 规则容易程序化 | 指标存在滞后性 |
| 适合短线和中短线交易 | 震荡行情容易反复打脸 |
| 能较快反映市场情绪 | 高频交易会被手续费和滑点侵蚀收益 |

> 一句话总结：**技术面策略适合回答"现在市场走势和情绪怎么样"，但不代表它知道资产真正值多少钱。**

## 三、统计套利策略：寻找价格关系的异常

### 3.1 什么是统计套利？

统计套利关注的不是某个资产本身涨不涨，而是：**两个或多个资产之间的价格关系是否偏离正常水平**。

> 核心流程：**发现价格偏离 → 建仓 → 等待关系回归 → 平仓**

统计套利不是简单预测上涨，而是判断价格关系是否异常。

### 3.2 配对交易示例

假设有两只银行股：招商银行和平平安安。它们都属于银行板块，长期走势有一定相关性。

| 状态 | 价格比值 | 策略信号 |
|---|---|---|
| 正常 | 招商银行 / 平安银行 ≈ 2 | 无 |
| 偏离 | 招商银行 / 平安银行 = 2.5 | 卖出招商 + 买入平安 |
| 回归 | 比值回到 2 | 平仓获利 |

> **配对交易赚的不是市场整体上涨的钱，而是两个资产之间价格关系修复的钱。**

### 3.3 跨市场套利示例

| 交易所 | ETH 价格 | 操作 |
|---|---|---|
| A 交易所 | 3500 美元 | 买入 |
| B 交易所 | 3520 美元 | 卖出 |

> 理论上可以赚取 20 美元价差。但现实中还要考虑：手续费、滑点、资金划转时间、交易所风险、成交速度、价格瞬间变化。

> **所以看到"套利"两个字，不要自动理解成稳赚。很多看起来存在的套利机会，真正执行后利润可能被成本吃掉。**

### 3.4 统计套利的优缺点

| 优点 | 缺点 |
|---|---|
| 理论基础较强 | 历史关系可能失效 |
| 可以做市场中性策略 | 对交易成本非常敏感 |
| 不完全依赖牛市 | 对执行速度和资金要求较高 |
| 更关注资产之间的相对关系 | 价格关系可能长期不回归 |

> 一句话总结：**统计套利不是无风险套利，它赌的是"异常关系会恢复正常"。但市场最残酷的地方在于，它可以长期不正常。**

## 四、AI 驱动策略：让机器学习复杂规律

### 4.1 什么是 AI 驱动策略？

AI 驱动策略是利用机器学习、深度学习、大模型等方法，从大量数据中寻找复杂规律。

它可能同时使用多种数据源：

| 数据类型 | 示例 |
|---|---|
| 市场数据 | 价格、成交量 |
| 财务数据 | 财报、指标 |
| 另类数据 | 新闻、研报、社交媒体情绪 |
| 宏观数据 | GDP、CPI、利率 |

然后模型输出预测结果：

```python
# AI 模型预测示例
预测("某只股票未来 5 天上涨概率是多少？")
预测("某只股票未来 20 天是否能跑赢指数？")
预测("当前新闻对公司股价是利好还是利空？")
```

### 4.2 常见 AI 模型

#### 随机森林 / XGBoost

这类模型适合处理**结构化数据**：

```python
# 特征工程
features = [PE, PB, ROE, 营收增长率, 净利润增长率, 
            过去20日收益率, 过去60日波动率, 成交量变化, 行业涨跌幅]

# 模型预测
预测("未来 20 个交易日是否跑赢指数")
```

#### LSTM / Transformer

这类模型更适合处理**时间序列数据**：

> 把股票过去 100 天的价格、成交量、技术指标输入模型，让它预测未来几天的走势。但金融市场和普通时间序列不一样，它噪声很大，规律也会变化。**很多模型在历史回测中表现很好，一到实盘就失效。**

#### NLP / 大模型

NLP 和大模型可以用来分析**文本信息**：

- 新闻情绪分析
- 公司公告解读
- 财报内容摘要
- 研报关键信息提取
- 舆情风险监控

### 4.3 AI 策略的风险

AI 策略听起来很高级，但不能迷信：

- **黑箱性强**：不容易解释为什么买入
- **容易过拟合**：历史数据表现好，实盘可能很差
- **数据质量要求高**：Garbage in, Garbage out
- **成本较高**：训练和计算资源消耗大
- **实盘差距**：回测和实盘效果可能差距很大
- **规律变化**：市场风格切换时模型可能快速失效

> **新手不要一上来就做 Transformer、强化学习、大模型炒股。更合理的学习路线是：先理解基本面因子和技术指标 → 再学习 Python 回测 → 然后做多因子策略 → 最后再尝试 XGBoost、NLP 和大模型。**

## 五、四种策略对比

| 策略 | 核心关注点 | 主要数据 | 适合周期 | 优点 | 缺点 | 适合人群 |
|---|---|---|---|---|---|---|
| 基本面策略 | 公司价值和经营质量 | 财报、行业、宏观 | 中长期 | 逻辑清晰，适合长期 | 数据滞后，短线能力弱 | 普通投资者、长期投资者 |
| 技术面策略 | 价格趋势和成交量 | K线、成交量、技术指标 | 短中期 | 反应快，容易程序化 | 震荡行情容易亏损 | 交易型投资者 |
| 统计套利 | 资产之间的价格关系 | 价差、相关性、协整关系 | 中短期 | 可做市场中性 | 执行难，成本敏感 | 专业量化团队 |
| AI 驱动策略 | 多源数据中的复杂规律 | 价格、财报、新闻、另类数据 | 不固定 | 能处理复杂关系 | 黑箱、过拟合、数据要求高 | 有编程和数据能力的人 |

## 六、四种策略不是互斥的

真正成熟的量化系统，往往不会只使用一种策略。更常见的组合方式是：

```
第一步：基本面筛掉垃圾公司
     ↓
第二步：技术面选择买入时机
     ↓
第三步：统计方法控制组合风险
     ↓
第四步：AI 模型辅助预测和优化权重
```

### 实战组合示例

| 步骤 | 策略 | 具体操作 |
|---|---|---|
| 1 | 基本面筛选 | ROE 高、负债低、利润增长稳定的公司 |
| 2 | 技术面择时 | 股价突破 60 日均线时买入 |
| 3 | 风险控制 | 控制行业分布，避免全部买同一个行业 |
| 4 | AI 辅助 | 机器学习模型预测未来收益，决定仓位大小 |

> **这比单纯依赖某一个指标要更稳健。**
