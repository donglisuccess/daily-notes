很多刚开始投资的人，挑选股票或基金时只关注一个指标：
> 收益率有多高？

某只基金过去三年涨了50%，看起来很好；另一只基金只涨了30%，似乎明显更差。

但投资不能只看最后赚了多少钱，还必须追问：

- 这50%的收益，中途最多亏过多少？
- 净值是平稳上涨，还是像坐过山车一样剧烈波动？
- 这只基金的高收益，是依靠稳定能力获得的，还是碰巧押中了某个行业？
- 当它下跌30%时，我还能不能继续持有？

真正成熟的投资者，不只看收益，还会观察：
> 为了获得这份收益，我承担了多少风险？

这就是“收益风险比”指标存在的意义。

在股票和基金投资中，最常见的三个收益风险比指标分别是：
- 夏普比率：Sharpe Ratio
- 卡玛比率：Calmar Ratio
- 索提诺比率：Sortino Ratio

它们都在研究“收益是否值得”，但衡量风险的方式并不相同。

为了方便理解，你可以先记住一句话：
> 夏普看整体稳不稳，卡玛看最惨亏多少，索提诺只看向下跌得狠不狠。

接下来，我们从零开始讲清楚这三个指标。

## 一、为什么只看收益率是不够的
假设现在有两只基金。

| 指标      |  基金A |  基金B |
| ------- | ---: | ---: |
| 过去一年收益率 |  20% |  12% |
| 最大回撤    | -40% | -10% |
| 年化波动率   |  35% |  12% |

只看收益率，基金A明显更高。

但基金A在这一年中，可能经历过这样的过程：

> 10万元 → 12万元 → 7.2万元 → 12万元

最终确实赚了20%，但中途曾经从12万元跌到7.2万元。

基金B可能是：
> 10万元 → 10.5万元 → 9.8万元 → 11.2万元

它只赚了12%，但整个过程更加平稳。

> 基金A多赚的8%，是否值得你承受-40%的最大回撤？

这没有统一答案。

风险承受能力强、投资期限长的人，可能愿意选择基金A；无法接受本金大幅亏损的人，可能更适合基金B。

但至少可以确定：

> 不能因为基金A收益率更高，就直接认定基金A更好。

收益风险比指标，就是帮助我们更公平地比较不同投资产品。

## 二、什么是收益风险比
收益风险比可以简单理解为：
> 收益风险比 = 获得的收益​ / 承担的风险

这个逻辑很像买东西时比较“性价比”。

例如：
- 产品A售价5000元，性能90分；
- 产品B售价3000元，性能80分。

产品A性能更强，但产品B可能性价比更高。

投资也是如此：
- 基金A收益更高，但风险特别大。
- 基金B收益略低，但风险小很多。

我们不能只问谁赚得多，还要问：
> 每承担一份风险，能够换回多少收益？

问题是，“风险”到底应该怎样定义？

有人认为净值上下波动就是风险；有人认为只有真正下跌才是风险；也有人最关心历史上发生过的最大亏损。

于是便形成了三个不同的指标。

| 指标    | 用什么代表风险 |
| ----- | ------- |
| 夏普比率  | 总波动率    |
| 卡玛比率  | 最大回撤    |
| 索提诺比率 | 下行波动率   |

它们不是互相替代的关系，而是从不同角度观察同一只基金。

## 三、夏普比率：每承担一份总波动，获得多少超额收益
### 1. 夏普比率是什么意思
夏普比率衡量的是：
> 投资者每承担一单位总波动风险，能够获得多少超过无风险收益的回报。

它的常见计算公式是：

```
夏普比率 = (投资组合年化收益率 - 无风险利率) / 投资组合年化波动率
```
公式看起来有些复杂，我们拆开理解。

#### 投资组合年化收益率
就是这只基金平均每年赚了多少。

例如过去三年的年化收益率为10%，可以理解为：
> 假设收益按照复利折算，平均每年大约增长10%。

年化收益率不是简单地把总收益除以年份，而是考虑了**复利**。

#### 无风险利率
无风险利率可以粗略理解为：
> 在不承担明显市场风险的情况下，投资者原本可能获得的收益。

实际计算中，可能参考短期国债收益率、货币市场利率等。

为什么要扣掉无风险利率？

因为假设低风险资产可以获得2%的收益，而某只高风险基金只获得3%，那么你承担了股票市场的大幅波动，却只多赚了1%。

这1%才是承担额外风险后获得的“超额补偿”。

因此：
```
超额收益=投资收益率−无风险利率
```

#### 年化波动率
波动率衡量的是：
> 基金收益围绕平均水平上下变化的程度。

波动率越大，说明基金净值越不稳定。

假设两只基金最终收益相同：

基金A：
```
+1%、+1%、+1%、+1%
```
基金B：
```
+10%、-8%、+12%、-6%
```
基金B的净值变化更加剧烈，因此波动率更高。

### 2. 夏普比率怎么计算
假设某只基金：
- 年化收益率：10%
- 无风险利率：2%
- 年化波动率：15%

那么：
```
夏普比率 = (10% - 2%) / 15% ≈ 0.53
```
这个0.53表示：
> 每承担1单位总波动风险，大约获得0.53单位超额收益。

夏普比率通常越高，说明风险调整后的表现越好。

但是这里一定要注意：

> 夏普比率越高，不代表绝对收益越高。

它代表的是“单位风险对应的超额收益更高”。

### 3. 两只基金的实战比较
假设有两只同类型的股票基金。
| 指标    | 基金A | 基金B |
| ----- | --: | --: |
| 年化收益率 | 16% | 12% |
| 无风险利率 |  2% |  2% |
| 年化波动率 | 28% | 12% |

基金A的夏普比率：
```
(16% - 2%) / 28% = 0.50
```
基金B的夏普比率：
```
(12% - 2%) / 12% ≈ 0.83
```
基金A的收益率更高，但基金B的夏普比率更高。

这意味着：
- 基金A赚得更多；
- 但基金A的净值波动也大得多；
- 基金B每承担一份波动风险，获得的超额收益更多。

所以，夏普比率给出的结论不是“基金B一定更赚钱”，而是：
> 基金B的收益获取过程更有效率。

### 4. 夏普比率最大的缺点
夏普比率把所有波动都当成风险。

无论基金是：
- 突然上涨10%；
- 还是突然下跌10%；

都会增加波动率。

但对投资者来说，上涨10%通常不是坏事。

假设某只基金经常大涨、很少大跌，它的波动率仍然可能较高，导致夏普比率受到影响。
