# 组件化思维

记录项目的整体分层方式，并帮助你快速复盘知识点。

## 为什么选择 Vue 3？

- 组合式 API 带来更高的复用性
- 基于 Vite 的极速冷启动与热更新
- TypeScript 友好，利于大型笔记仓库扩展

## 推荐目录规范

1. 在 `src/pages` 中创建业务线目录，例如 `前端/`、`后端/`
2. 二级目录可以继续拆分主题，如 `组件化`、`工程化`
3. 文件名尽量语义化，方便在左侧目录中快速定位

## 常用命令

```bash
pnpm install
pnpm dev
```

## 板块拆分

> 将页面视为一个数据流动的容器，每个区域都是独立组件。

| 模块 | 说明 |
| ---- | ---- |
| Sidebar | 自动读取 Markdown 目录结构 |
| Viewer | 渲染选中文章内容 |
| Toc | 根据标题生成大纲 |
